It uses [GlueSQL-js](https://github.com/gluesql/gluesql-js).

This demo is not only made using `GlueSQL-js`, but also it uses `GlueSQL-js` for state management (something like `redux`).

### Installation
```
npm install
```
### Start
```
npm start
```

### Test
```
npm test
```
